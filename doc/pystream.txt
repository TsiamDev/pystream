Compiler
========

Architechture
-------------

The compiler analyizes and optimizes the code all at once.  Incremental compilation is not performed.


* Parsing
* Analysis
* Optimization
* Output


Parsing
-------

* No traditional parser.
* Extracts program definition from memory.
* Uses Python as a "preprocessor."
* Decompilation
	* Disassebly
	* Structural analysis
	* Destacking
		* Bizare Pythonisms

Analysis
--------

* BDD-based analysis.
* Datalog algorithim spesification.
* Two phases
	+ FI CFA "approximation"
	+ FS CPA "refinement"

* Generic engine for term rewriting
